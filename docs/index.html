Claude Code is a CLI. ArmyClaw is the operations center on top of it — orchestrating dozens of long-running agents from a single subscription. Three things separate it from every other wrapper:

15–36×cheaper

Subscription, never API

Spawns the local claude binary using your claude login. Never sets ANTHROPIC_API_KEY. Your Pro/Max plan covers every agent, every group member, every routine, every Telegram message — flat fee.

∞parallel agents

Real subprocess battalion

Each chat is its own long-running agent. Switching tabs never kills a stream. Group chats spawn one agent per persona member, all orchestrated by a single supervisor. Watch them brainstorm in real time.

3surfaces

Web · Telegram · Terminal

Drive any agent from a clean web SPA, your phone via a Telegram bot you control, or the inbuilt terminal panel. All three surfaces share the same agent fleet — start a task on your laptop, finish it from your phone.

∞models

Model-agnostic backend

Two env vars (ANTHROPIC_BASE_URL + ANTHROPIC_AUTH_TOKEN) and every agent routes through GLM-4.7, Kimi K2.5, OpenRouter, DeepSeek, GPT-5, or local Ollama — fully offline if you want. Same orchestrator, any brain.

∞characters

Character.AI features built in

Persistent personas, group "Rooms" where 2–5 agents banter with each other peer-to-peer, cross-chat memory, voice input. The roleplay/companion stack and the dev workflow run on the same orchestrator.

Get an army running in 90 seconds.

Three commands. One Python dependency. Your claude login already exists; ArmyClaw just orchestrates around it. 

# 1. Install the claude CLI and sign in (one-time).
npm install -g @anthropic-ai/claude-code
claude login                # browser → your Pro/Max account

# 2. Clone + install (one Python dep).
git clone https://github.com/atmostai-online/armyclaw.git armyclaw
cd armyclaw
pip3 install --user "websockets>=12,<17"

# 3. Run.
python3 server/cc-server.py
# → http://localhost:8765/
